# -
仿真Linux系统

【文件夹说明】
- jre：包含Java运行环境及必要库文件（请不要删除或移动）
- input：存放测试数据（作业请求文件 jobs-input.txt 及各作业指令文件 *.txt）
- output：程序运行时自动生成日志文件（ProcessResults-xxx-B.txt），初始为空
- RunProcess.exe：主程序可执行文件，双击即可运行

【安装步骤】
1. 一定要确保 jre、input、output 三个子文件夹与 RunProcess.exe 在同一级目录下。jre自行解压为文件夹
2. 若需使用自己的测试数据，请替换 input文件夹中的内容，但需保持文件格式正确。

【使用方法】
1. 双击 RunProcess.exe 启动程序。
2. 点击界面顶部的“执行”按钮，开始仿真。
3. 可随时点击“暂停”/“继续”控制调度过程。
4. 点击“保存”按钮可将当前调度日志保存到 output 文件夹。
5. 点击“实时”按钮可在当前时刻插入一个实时作业（20条指令）。
6. 所有作业完成后，可关闭窗口，程序会自动保存完整日志。

【注意事项】
-一定要确保 jre、input、output 三个子文件夹与 RunProcess.exe 在同一级目录下。
-必须手动点击保存按钮才会保存日志文件。
- 若程序无法运行，请检查是否缺少 jre 文件夹或杀毒软件误删。
- 如需查看源代码，请打开 src 子文件夹中的工程文件（IntelliJ IDEA 项目）。
